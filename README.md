# Student Performance Analysis

This project analyzes a student performance dataset using Python and pandas.

## Tools Used
- Python
- pandas
- NumPy
- Matplotlib
- Seaborn

## Key Work
- Data cleaning and handling missing values  
- Feature creation (TotalScore, Z-Score)  
- Visual analysis using boxplots, bar charts, heatmaps, and pie charts  
- Study of factors like test preparation, parents’ education, and activities  

## Files
- `student_performance_analysis.ipynb` → Main notebook  
- `student_performance_analysis.pdf` → Final report
